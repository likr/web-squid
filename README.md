# web-squid

## Development

### Requirements
* Node.js & NPM

* Grunt - Run following command.

	```shell
 	npm install -g grunt-cli
	```

* Grunt Packages - Run following command at project directory.

	```shell
	npm install
	```

### Start Development

* Build

	```shell
	grunt compile
	```

* Watch

	```shell
	grunt
	```
